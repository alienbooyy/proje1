# proje1-pos

Yerel ağda çalışan restoran POS sistemi (offline).

## Özellikler (MVP)
- Masalar ve adisyon yönetimi
- Ürün yönetimi
- Sipariş oluşturma (tablet/web)
- Gün sonu raporu (tarih aralığı)

## Kurulum

```bash
npm install
npm start
```

Uygulama varsayılan olarak `http://localhost:3000` adresinde çalışır.

## Notlar
- Veritabanı SQLite kullanır ve `data/pos.db` dosyasına yazılır.
- Yazdırma ve gelişmiş raporlama için ek geliştirmeler yapılacaktır.
